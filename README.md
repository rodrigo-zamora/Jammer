## Models
- **User**
    - UUID *string*
    - firstName *string*
    - lastName *string*
    - imageURL *string*
    - email *string*
    - password *string*
    - createdAt *date*
    - updatedAt *date*
    - subscription *string*
    - list *string[]*
- **List**
    - UUID *string*
    - name *string*
    - movieUUID *string[]*
    - isShared *boolean*
    - sharedWith *string[]*
- **Movie**
    - UUID *string*
    - title *string*
    - poster *string*
    - year *number*
    - synopsis *string*
    - rating *number*
    - duration *string*
    - genres *string[]*
    - director *string[]*
    - cast *string[]*
- **Subscription**
    - UUID *string*
    - paymentUUID *string*
    - subscriptionType *string, enum(free, premium)*
- **Payment**
    - UUID *string*
    - paymentType *string, enum(paypal, creditCard)*
